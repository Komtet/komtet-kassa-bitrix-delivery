komtet_kassa_bitrix
===============================

Чтобы развернуть окружение и потестировать плагин в docker контейнере:
1. создать в дирректории docker_env папку php, скопировать в нее текущий установщик Bitrix, установить на нее права 777
2. docker run -d --name mysite -p 80:80 -p 2222:22 -p 443:443 -p 8893:8893 -p 8894:8894 -v `pwd`/php:/home/bitrix/www -e BVAT_MEM=524288 -e TIMEZONE="Asia/Novosibirsk" constb/bitrix-env
   подставив свои значения в параметры названия контейнера, путей, таймзоны
3. перейти в браузере на localhost и выполнить установку CMS, указать существующую базу данных bitrix, нового пользователя root без пароля
4. по ссылке https://www.1c-bitrix.ru/bsm_register.php получить тестовый лицензионный ключ
5. сохранить лицензионный ключ в настройках Битрикс, установить и настроить плагин
